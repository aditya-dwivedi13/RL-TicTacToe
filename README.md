# RL-TicTacToe
A research-oriented reinforcement learning project that explores self-play, neural networks, and custom learning algorithms through the game of Tic-Tac-Toe using PyTorch.

## About
This project is a modular implementation of Tic-Tac-Toe built as a playground for experimenting with Reinforcement Learning and Neural Networks.

The primary goal is not simply to create an AI that can play Tic-Tac-Toe, but to build and understand the complete reinforcement learning pipeline from scratch. To achieve this, the project separates the game engine, AI agent, and neural network into independent modules, allowing different learning algorithms to be tested without modifying the core game logic.
The project aims to experiment with custom models and understand their ability of finding the optimal strategies that are well known in TIC-TAC-TOE

## Version 1
**This version focuses on building the complete software architecture and integrating an untrained Convolutional Neural Network (CNN) into the game.** Although the AI can already play against a human, its decisions are currently based on randomly initialized network weights. Future versions will train this network through self-play and reinforcement learning, followed by experiments with different learning strategies and custom algorithmic ideas.
